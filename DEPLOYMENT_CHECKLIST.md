# 📋 Route Checklist - QA Before Deployment

## Before Deploying to GitHub

Test each route in development mode:

```bash
cd web && npm run dev
```

Check each page for:
- ✅ Loads without errors
- ✅ CSS/images display correctly
- ✅ Interactive features work (carousels, animations, etc.)
- ✅ Responsive on mobile/desktop
- ✅ No console errors
